# WallpaperSwapper
    This software is WIP althought tested and usable

## Installation
```bash
$ git clone git@github.com:hajekad/WallpaperSwapper.git
$ cd WallpaperSwapper
$ make
$ ./WaSp
```

## Configuration
Open the config.hpp and configure the program's behavior
